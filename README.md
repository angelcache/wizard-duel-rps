# 🪄 Wizard Duel RPS

A fun, ✨magical✨ twist on Rock-Paper-Scissors!  
Cast spells, battle your opponent, and be the first to win 5 rounds.

---

## 🔊 Features

- Animated health bars  
- Optional background music  
- Cute magical girl vibes
